# Project 2 – Mission Demolition

Liam Johnston  

This project is a physics-based game where the player launches projectiles to destroy structures.

The goal of this project is to understand Unity’s physics system and how objects interact through forces and collisions.

## Gameplay

The player uses a slingshot mechanic to launch projectiles at stacked structures. The objective is to knock everything down using a limited number of shots.

## What I Implement

- Slingshot launch system using mouse input  
- Rigidbody-based physics interactions  
- Destructible structures made from stacked objects  
- Projectile system with multiple attempts  
- Camera that follows the projectile  
- Reset system for retrying  
- Win and lose conditions  

## Technologies Used

- Unity  
- C#  
- Rigidbody physics system  
- Camera scripting  

## Notes

This project focuses mainly on physics behavior and interactions rather than UI or polish.

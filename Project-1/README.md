# Project 1 – Apple Picker

Liam Johnston  

This project is a Unity arcade-style game where the player moves a basket to catch falling apples.

The main goal of this project is to get comfortable with Unity basics like object spawning, collisions, and UI.

## Gameplay

Apples fall from the top of the screen and the player moves left and right to catch them. Missing apples or hitting hazards results in losing baskets.

The game increases in difficulty over multiple rounds.

## What I Implement

- Start menu with a working start button  
- Basket system with multiple lives (modified from base version)  
- Round progression (increasing difficulty each round)  
- Game over system with restart functionality  
- Hazard object (branch) that ends the game on contact  
- Randomized apple and hazard spawning  

## Technologies Used

- Unity  
- C#  
- Unity UI system  
- Basic collision and physics  

## Notes

This project focuses on learning core systems rather than building something complex.

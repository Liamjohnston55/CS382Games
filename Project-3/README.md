# Project 3 – Space SHMUP

Liam Johnston  

This project is a 2D space shooter where the player controls a ship, shoots enemies, and tries to survive as long as possible.

This project combines multiple systems into a more complete game.

## Gameplay

The player moves a spaceship horizontally and shoots upward at incoming enemies. Enemies spawn over time and increase in difficulty.

The goal is to survive and get the highest score possible.

## What I Implement

- Player movement using keyboard input  
- Shooting system with instantiated projectiles  
- Enemy spawning system with scaling difficulty  
- Collision detection (player, bullets, enemies)  
- Health / life system  
- Score tracking  
- Game over system  
- Restart functionality  

## Technologies Used

- Unity  
- C#  
- Input system  
- Object instantiation  
- Collision system  

## Notes

This project brings together multiple systems and focuses more on actual gameplay rather than isolated mechanics.

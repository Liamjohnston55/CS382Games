# Game Development Projects (Unity)

Liam Johnston  

This repository contains a set of game development projects built using Unity and C#. These projects focus on learning and implementing core systems used in real-time games.

Each project builds on the previous one, starting from basic mechanics and moving toward more complete gameplay systems.

## Projects Included

### Project 1 – Apple Picker
A basic arcade-style game focused on object spawning, collisions, and UI.

### Project 2 – Mission Demolition
A physics-based game using rigidbody interactions and projectile mechanics.

### Project 3 – Space SHMUP
A 2D shooter that combines movement, shooting, enemies, scoring, and game state systems.

## Skills Demonstrated

- Unity game development  
- C# scripting  
- Physics and collision systems  
- Object instantiation and spawning  
- Game state management  
- UI integration  

## Structure

Each project is contained in its own folder and includes its own README with more detailed explanations.

## Notes

These projects are built from structured assignments, but all gameplay logic, modifications, and additional features are implemented by me.

The goal of this repository is to show progression from basic Unity concepts to more complete game systems.

# Wild Acentions

Liam Johnston  

Wild Acentions is an open-world style Unity game where the player explores, gathers resources, and progresses through a character-based system.

This project focuses on building multiple interconnected systems and getting them to work together in a consistent way.

## Gameplay

The player controls a character that can move, jump, and interact with objects in the world.

Resources can be harvested from environmental objects, each with their own health and models. Collected items are stored in an inventory system.

The player has health, can level up, and is affected by carried weight, which slows movement similar to survival-style games.

Game progress is saved and loaded, allowing persistent progression.

## What I Implement

- Player movement system (including jumping)  
- Inventory system with item storage and management  
- Item system with different resource types  
- Weight-based system that affects player movement  
- Harvestable objects with health and destructible behavior  
- Health system for both player and world objects  
- Character leveling system  
- Save/load system for persistent game state  
- UI system including health bars and inventory display  
- General world interaction systems
